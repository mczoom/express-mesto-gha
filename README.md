# «Место» (backend)
Сервис для публикации фотографий.

Серверная часть проекта [Место](https://github.com/mczoom/react-mesto-auth).


## Использованные технологии:
- Node.js
- Express.js
- MongoDB



### Инструкция по запуску
#### `npm run start`

запускает сервер

#### `npm run dev`

запускает сервер с hot-reload
